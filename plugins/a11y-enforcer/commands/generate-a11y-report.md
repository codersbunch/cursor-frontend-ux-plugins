# Generate Accessibility Report

Run automated accessibility audit and produce actionable report.

## Steps

1. Run axe-core against all pages/components
2. Categorize violations by WCAG criterion and severity
3. Group by component for easier fixing
4. Generate report with violation count by severity
5. Output to `accessibility-report.md`
